# 📚 Graph Guide

Graphs used to feel confusing to me—especially BFS, DFS, cycle detection, topological sort, and knowing which algorithm to use in different situations.

So I decided to learn them one by one and document everything in a simple, interview-friendly way.

This repository includes:

- 📖 Easy-to-understand explanations
- 💻 Clean Java implementations
- 📝 Step-by-step dry runs
- ⚡ Time & Space Complexity
- 🎯 Interview questions and tips
- 🔗 LeetCode practice problems
